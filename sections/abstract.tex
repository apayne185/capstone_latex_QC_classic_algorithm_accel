


\begin{abstract} 
As quantum processing units (QPUs) evolve as specialized accelerators, integrating quantum kernels into High Performance Computing (HPC) environments has become an emerging frontier in computational science. Quantum-classical hybrid systems combine the strength of both paradigms to overcome hardware limitations in qubit coherence and error rates. An example is the Variational Quantum Eigen-solver (VQE), which offloads the computationally intensive classical optimization loop to conventional computing resources, while QPUs handle the execution of quantum circuits and the measurement of resulting expectation values. There is a lack of existing unified middleware to bridge low-latency QPU control with multi-node HPC infrastructures, so for that reason the development of this project will focus on three main objectives: designing an intuitive API to abstract the cloud-based QPU backend's complexity, the development of HPC connectors using MPI/CUDA to parallelize classical subroutines, and a performance analysis of the framework using VQE workflows. The stack's evaluation will occur on a multi-node cluster, with the objective of achieving a 2-3x speedup in end-to-end wall-clock time as compared to single-node and non-optimized implementations. This work aims at a robust, open-source framework that facilitates near-linear scalability in hybrid simulations in fields such as materials science or complex optimization. 
\end{abstract}

Keywords: Quantum Systems, Hybrid Systems, Variational Quantum Eigen-solver, High-Performance Computing, Algorithm Acceleration, MPI, CUDA, Software Middleware.
