


\begin{abstract} 
As quantum processing units (QPUs) evolve as specialized accelerators, integrating quantum kernels into High Performance Computing (HPC) environments has become an emerging frontier in computational science. Quantum-classical hybrid systems combine the strength of both paradigms to overcome current hardware limitations in qubit coherence and error rates. An example is the Variational Quantum Eigen-solver (VQE), which offloads the computationally intensive optimization loop to classical computing resources, while the QPUs handle the execution of quantum circuits and the measurement of resulting expectation values. There is a lack of existing unified middleware to bridge low-latency QPU control with multi-node HPC infrastructures, so for that reason the development of this project will focus on three main objectives: designing an intuitive API to abstract the cloud based QPU backend's complexity, the development of HPC connectors using MPI/CUDA in order to parallelize classical subroutines, and a performance analysis of the framework using VQE workflows. The stack's evaluation will occur on a multi-node GPU cluster, with the objective of achieving a 2--3x speedup in end-to-end wall-clock time as when compared to a single-node and non-optimized implementations. This work aims at the development of a robust, open source framework that will facilitate near-linear scalability in hybrid simulations for future usage in fields such as materials science or complex optimization. 


\noindent\textbf{Keywords:} Quantum Systems, Hybrid Systems, Variational Quantum Eigen-solver, High-Performance Computing, Algorithm Acceleration, MPI, CUDA, Software Middleware.
\end{abstract}
